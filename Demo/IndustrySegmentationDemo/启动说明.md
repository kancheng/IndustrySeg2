# IndustrySegmentationDemo 启动说明

## 设置为启动项目

IndustrySegmentationDemo 已配置为默认启动项目。在 Visual Studio 中：

1. **自动启动**：打开解决方案后，IndustrySegmentationDemo 会自动设置为启动项目
2. **手动设置**：如果未自动设置，请右键点击 `IndustrySegmentationDemo` 项目，选择"设为启动项目"
3. **运行**：按 `F5` 或点击"启动"按钮运行程序

## UI 界面功能

IndustrySegmentationDemo 提供了完整的图形界面，包含以下功能：

### 配置区域
- **模型文件选择**：选择 ONNX 模型文件（如 sd900.onnx）
- **图片路径选择**：支持单文件或批量处理模式
- **输出目录选择**：选择结果保存位置
- **参数调整**：
  - Confidence（检测置信度）
  - Pixel Confidence（像素置信度）
  - IoU（交并比）

### 操作区域
- **开始检测**：开始处理图片
- **停止检测**：取消正在进行的处理
- **打开输出文件夹**：快速打开结果目录

### 显示区域
- **左侧**：检测结果预览（实时显示处理后的图片）
- **右侧**：
  - 统计信息（总处理数、NG/OK 数量）
  - 处理进度
  - 日志输出

## 使用流程

1. 选择模型文件（默认会自动查找项目中的 sd900.onnx）
2. 选择图片文件或目录
3. 调整参数（可选，使用默认值即可）
4. 点击"开始检测"
5. 查看处理结果和统计信息

## 注意事项

- 首次运行时会自动查找项目根目录下的默认模型和测试图片
- 批量处理时，处理结果会实时更新显示
- 可以随时点击"停止检测"按钮取消正在进行的处理
- 处理结果会自动保存到输出目录，文件名格式：`原文件名_NG.png` 或 `原文件名_OK.png`
